n8n: AI Workflow Automation

Memory Map
💡

The Big Idea

n8n = Zapier, but open-source, self-hosted, free, and AI-native. Drag-and-drop visual workflows that connect LLMs to your business processes. A huge number of AI use cases don't need custom code — they need automation. n8n builds in an afternoon what takes hours to code.

🏆 Three Reasons n8n Wins for AI

🔓
Open Source & Self-Hosted
Runs on your server. Your data never leaves your network. Free and unlimited.
🤖
Native AI Nodes
Purpose-built nodes for agents, RAG, classification, extraction, sentiment. Not generic HTTP wrappers.
🏠
Ollama Integration
Connect to local LLMs. Fully private AI automation stack. Zero API costs per request.

🧠 The AI Nodes

🤖 AI Agent
Full agent with tools — makes decisions, calls tools, iterates. LangGraph-level capability in a visual node.
💬 Chat Model
Connect to any LLM (OpenAI, Anthropic, Ollama, HuggingFace). Configure model, temperature, system prompt visually.
🗄️ Vector Store
Connect to Pinecone, Chroma, Qdrant. Build RAG workflows without writing retrieval code.
🏷️ Text Classifier
Categorize text into predefined categories. Support tickets, feedback, emails.
😊 Sentiment Analysis
Positive, negative, neutral. Customer feedback, reviews, social media monitoring.
📋 Information Extractor
Extract structured data (names, dates, amounts) from unstructured text. Like output parsing, but visual.

🚀 Getting Started (30 seconds)

  1. Run n8n

    npx n8n or docker run -it --rm -p 5678:5678 n8nio/n8n

  2. Open browser

    Go to http://localhost:5678. Visual canvas ready to go.

  3. Connect an LLM

    Paste an OpenAI key or point to Ollama at http://localhost:11434.

  4. Browse templates

    Don't start from scratch. Clone an AI template and modify it. Hundreds available.

🧪 Quick Quiz

What makes n8n different from Zapier for AI workflows?

Open-source, self-hosted (data stays on your server), native AI nodes (agents, RAG, classifiers), and local LLM support via Ollama. Zapier is cloud-only with basic AI actions.

What is the "privacy stack"?

n8n + Ollama. Workflows run on your server, LLM runs on your server, data never leaves your network, zero API costs. Ideal for regulated industries.

Name the 6 AI-specific nodes in n8n.

AI Agent (autonomous decisions), Chat Model (LLM connection), Vector Store (RAG), Text Classifier, Sentiment Analysis, and Information Extractor.

When should you code instead of using n8n?

When you need complex AI logic (self-correction, multi-agent), custom UI, high performance, or the AI is core to your app's logic. n8n is for automation, not for building app features.

How does n8n connect to Android development?

Your app calls a webhook. n8n handles the AI automation (classify feedback, draft responses, curate content) and sends results back via API. Your app doesn't know n8n exists.

📌 Key Takeaways

  • n8n = open-source Zapier with native AI. Visual drag-and-drop workflows. Self-hosted. Free.
  • 6 AI nodes: Agent, Chat Model, Vector Store, Text Classifier, Sentiment Analysis, Information Extractor.
  • Privacy stack: n8n + Ollama = fully private AI automation. Data never leaves your network.
  • Example workflows: email classification, Slack RAG bot, content pipeline — each built in 30-60 minutes.
  • n8n beats Zapier/Make for AI: open-source, self-hosted, native AI nodes, local LLM support.
  • Don't start from scratch. Browse n8n's template library. Clone, modify, run.
  • Code instead when you need complex AI logic, custom UI, high performance, or deep app integration.
  • For Android: your app calls a webhook. n8n automates the AI backend. App fetches results via API.
