The Big Idea

AI only understands numbers. Every modality — images, audio, text — must be converted into embeddings by specialized encoders. Once everything is numbers in the same format, models can reason across all modalities together.

🌍 The Core Analogy: UN Interpreters

🖼️ Images speak in pixels
🔊 Audio speaks in waveforms
🤖 Encoders translate everything into embeddings
📝 Text speaks in characters
🎬 Video = images + audio + time

Like UN interpreters: each encoder specializes in one "language" (modality) and translates into a common "diplomatic language" (embeddings).

🧩 Core Concepts

🔢
Everything Is Numbers
Computers don't "see" or "hear" — they process numbers. A photo is ~6.2M numbers (RGB pixels). One second of audio is 44,100 numbers.
🧠
Encoders
Specialized neural networks that convert raw data (pixels, waveforms, tokens) into compact, meaningful embeddings.
📊
Embeddings
Dense vectors of numbers (e.g., 768 floats) that capture the meaning of input data. Similar inputs produce similar embeddings.
📈
Spectrograms
The clever trick: convert sound into a 2D image (time × frequency). This lets audio AI reuse vision techniques.
🧩
Vision Transformers (ViT)
Chop image into 16×16 patches, treat each patch like a "word token," apply same Transformer architecture as LLMs.
📦
Pre-Trained Encoders
You never train from scratch. Use encoders already trained on billions of examples (ImageNet, LAION-5B, 680K hours of audio).

📖 Key Terms

Encoder
Neural network that converts raw modality data into numerical embeddings.
Embedding
A dense vector of numbers capturing the meaning of an input — the "common language" all modalities share.
Pixel
A single colored dot in an image, defined by 3 numbers (Red, Green, Blue, each 0-255).
Spectrogram
A 2D image of sound: time on x-axis, frequency on y-axis, brightness = volume.
Mel Spectrogram
A spectrogram with frequency adjusted to match human pitch perception.
Vision Transformer (ViT)
Modern vision encoder that splits images into patches and processes them with the Transformer architecture.
CNN (Convolutional Neural Network)
Older vision encoder that slides filters across images to detect patterns from edges to objects.
Tokenization
Splitting text into chunks (tokens), each assigned a numeric ID from a vocabulary.
Pre-trained Encoder
An encoder already trained on massive datasets (ImageNet, LAION, AudioSet) — ready to use off the shelf.

❓ Quick Quiz

Why does AI need to convert images, audio, and text into numbers?

Because computers only understand numbers. Every modality must be converted into numerical embeddings before any AI model can process it.

How many numbers represent a standard HD image?

About 6.2 million (1920 × 1080 pixels × 3 RGB values each).

What's the clever trick that makes audio processing similar to image processing?

Converting sound into a spectrogram — a 2D image of sound (time × frequency). This turns "how does a machine hear?" into "how does a machine see?"

What is a Vision Transformer (ViT) and why does it matter?

ViT chops images into patches, treats each patch like a word token, and uses the same Transformer architecture as LLMs. It's the dominant approach in GPT-4V, Claude, and Gemini — unifying vision and language under one architecture.

What is the common output format of all encoders?

A sequence of embedding vectors — dense lists of numbers that capture meaning. Once all modalities share this format, models can compare, combine, and translate between them.

📌 Key Takeaways

  • AI only understands numbers — every modality (images, audio, text) must be encoded into numerical embeddings.
  • Encoders are like UN interpreters: each specializes in one modality but translates into a common language (embeddings).
  • Images → pixels → vision encoder (ViT) → embeddings. Audio → waveform → spectrogram → encoder → embeddings.
  • Spectrograms turn sound into images, so audio AI can reuse vision techniques.
  • Vision Transformers (ViT) use the same architecture as LLMs — patches are to images what tokens are to text.
  • You almost never train encoders from scratch — pre-trained encoders are available off the shelf.
