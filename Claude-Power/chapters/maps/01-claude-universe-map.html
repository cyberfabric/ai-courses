The Big Idea

Claude is one brain with five bodies. Every product runs the same models — the difference is the interface and the tools available. Pick the right body for the job.

🗺️ The Ecosystem

💬
Claude.ai
The conversational interface. Best for brainstorming, writing, research, learning, and quick prototypes via Artifacts.
⌨️
Claude Code CLI
The developer's power tool. Reads your files, runs commands, manages git, automates workflows — all from the terminal.
🖥️
IDE Extensions
Claude Code inside VS Code and JetBrains. Inline diffs, @-mention files, quick questions without leaving the editor.
🔌
Claude API
Programmatic access. Build Claude into your apps with Messages API, streaming, tool use, vision, and prompt caching.
🤖
Agent SDK
Build autonomous agents. Define tools and goals — Claude plans steps, uses tools, observes results, and keeps going.
🔗
MCP
Model Context Protocol — the integration layer. Connects Claude to GitHub, Slack, databases, and 6,000+ tools.

🪐 Beyond the Dev Stack

🧑‍💼
Claude Cowork (Jan 2026)
Agentic assistant for business tasks — calendar, email, docs, research, MCP connectors. The teammate-shaped Claude.
🎨
Claude Design (Apr 17 2026)
Anthropic Labs research preview running on Opus 4.7. Generates design systems, prototypes, slide decks. Lives as a plugin in Cowork.

📖 Key Terms

Artifacts
Interactive outputs in Claude.ai that live outside the chat — code, docs, diagrams, mini-apps.
MCP (Model Context Protocol)
Open standard that lets Claude connect to external tools and services — GitHub, Slack, databases, and more.
CLAUDE.md
Project-level instruction files that Claude Code reads automatically — your project's persistent context.
Agent SDK
Framework for building autonomous agents: define tools and goals, Claude orchestrates the execution.
Extended Thinking
API feature where Claude reasons step-by-step before responding — better for complex problems.
Prompt Caching
API feature that reduces costs dramatically by caching repeated context across requests.
Tool Use (Function Calling)
API capability where Claude calls functions you define, acts on results, and keeps going in a loop.
--print mode
Claude Code flag for headless/scriptable output — perfect for CI/CD pipelines and automation.

⚠️ Common Mistakes

❌
Claude.ai for coding
Copy-pasting code into the chat is slow and error-prone. Use Claude Code — it reads your files directly.
❌
API for one-off questions
Writing a script to ask a single question? Just use Claude.ai or Claude Code instead.
❌
Always reaching for Opus 4.7
Opus 4.7 is the flagship, but Sonnet 4.6 handles 90% of tasks at 60% of the price — and faster. Match the model to the task.
❌
Ignoring CLAUDE.md
Without project instructions, Claude starts from scratch every time. Set it up once, benefit forever.

🎯 Quick Quiz

You want to explore a new API design with pros and cons. Which product?

Claude.ai — it's best for brainstorming, analysis, and weighing tradeoffs in conversation.

You need to refactor 5 files and run the test suite. Which product?

Claude Code (CLI) — it can read files, make edits, run tests, and verify results all in one flow.

You're building a feature that auto-summarizes support tickets. Which product?

Claude API — you need programmatic access to integrate summarization into your app's backend.

You want to quickly ask about a function while editing in VS Code. Which product?

IDE Extension — select the code, ask in the Claude panel, get an answer without leaving your editor.

What's the key difference between an API call and an Agent SDK agent?

An API call is one prompt → one response. An Agent SDK agent gets a goal, plans steps, uses tools, observes results, and keeps going autonomously until the goal is met.

A teammate asks “does Claude do calendar and email work?” — where do you point them?

Claude Cowork — Anthropic’s agentic assistant for business tasks (calendar, email, docs, research). Claude Design ships as a plugin inside it for prototype/slide generation.

📌 Key Takeaways

  • Claude has 5 dev-facing surfaces: Claude.ai, Claude Code CLI, IDE Extensions, API, and Agent SDK — all sharing the same models.
  • Use Claude.ai for thinking (brainstorm, write, research). Use Claude Code for doing (edit files, run commands, automate).
  • Opus 4.7 (April 16, 2026) is the new flagship — same $5/$25 price as 4.6, jumped from 80.8% → 87.6% on SWE-bench Verified, adaptive thinking only.
  • Sonnet 4.6 (79.6% SWE-bench) is your daily driver — fast, capable, and 40% cheaper than Opus. Reach for Opus 4.7 on hard agentic loops.
  • MCP connects Claude to 6,000+ external tools — it's how Claude interacts with GitHub, Slack, databases, etc.
  • Claude Cowork (Jan 2026) and Claude Design (Apr 2026) sit beyond the dev stack — agentic business assistant and design-prototype generator.
  • The biggest power user mistake is using the wrong product for the task. Match the tool to the job.
