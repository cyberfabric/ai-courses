The Big Idea

Managed Agents let you send Claude off to work in the cloud — no laptop open, no terminal running, no babysitting. Define what to do, fire the task, and get results when it's done.

🏢 The Core Analogy

🪑
Claude Code
A brilliant colleague sitting next to you. Interactive, real-time, needs your laptop.
🔌
Claude API
Claude embedded in your product. You build the UI and manage the agent loop yourself.
🏢
Managed Agents
That colleague working from their own office, sending you a report when they're done.

📖 Key Terms

Agent
A reusable definition: model + system prompt + tools + MCP servers. Create once, use across many sessions.
Environment
A cloud container template with packages, network rules, and mounted files — the agent's workspace.
Session
A running instance combining Agent + Environment. Each session is independent and isolated.
Events (SSE)
Server-Sent Events streaming what the agent is doing: messages, tool calls, results, completion.
agent_toolset_20260401
The built-in tool bundle: Bash, file ops, web search, web fetch, MCP — all in one.
MCP Servers
External service integrations (GitHub, Slack, databases) the agent can use autonomously.
Fire-and-Forget
Start a session and walk away — results persist even if your connection drops.

⚡ API Quick Reference

Step Endpoint What It Does
1 POST /v1/agents Create agent definition (once)
2 POST /v1/environments Create environment (once)
3 POST /v1/sessions Start a session (per task)
4 POST /v1/sessions/{id}/events Send a user message
5 GET /v1/sessions/{id}/stream Stream events (SSE)

🎯 Power User Use Cases

🔍
Automated Code Review
Trigger on every PR. Agent clones, tests, analyzes diff, posts review comments. Zero human effort for first pass.
📊
Scheduled Reports
Daily code quality audit: coverage trends, dependency health, dead code. Report in Slack before standup.
🔬
Research Tasks
Evaluate 5 libraries, test compatibility, deliver a comparison matrix — all while you focus on other work.
📦
Data Processing
Batch clean/transform/load 500 CSV files. Agent scripts its way through while you write business logic.

Multi-Agent Teams
Chain agents together: backend analyst + frontend analyst + infra checker. Coordinator merges all reports.

🚀 Getting Started: 4 Steps

  1. Install & Configure

    pip install anthropic + set ANTHROPIC_API_KEY.

  2. Create Agent + Environment

    Define model, system prompt, tools, and container config. Create once, reuse forever.

  3. Start a Session

    Combine Agent ID + Environment ID, send a user message with the task.

  4. Stream or Walk Away

    Watch events in real-time or check back later — sessions persist regardless.

🧪 Quick Quiz

What are the four core objects in Managed Agents?

Agent (definition), Environment (workspace), Session (running instance), Events (communication stream).

When should you use Managed Agents vs Claude Code?

Use Claude Code when you need to watch and interact. Use Managed Agents for fire-and-forget tasks, long-running work (30min+), and scheduled/recurring jobs.

What does agent_toolset_20260401 include?

Bash, file operations, web search, web fetch, and MCP server access — all bundled into a single tool type.

What happens if your connection drops during a session?

Nothing — sessions persist. You can reconnect later and retrieve all events that happened while you were away.

What's the "power move" for complex analysis tasks?

Chain agents together — spin up multiple specialist agents in parallel (backend, frontend, infra) and aggregate their reports with a coordinator script.

🎯 Key Takeaways

  • Managed Agents run Claude on Anthropic's cloud — no laptop, no terminal, no babysitting.
  • Four core concepts: Agent (who) → Environment (where) → Session (go) → Events (status).
  • Create agents and environments once, reuse across unlimited sessions.
  • Built-in tools (Bash, files, web, MCP) mean your agent can do almost anything autonomously.
  • Best for: long-running analysis, scheduled reports, automated PR review, batch processing.
  • Sessions persist even if your connection drops — true fire-and-forget.
  • Cost is mostly tokens (same as API) + a tiny $0.08/hr runtime fee. Default to Sonnet 4.6; pay for Opus 4.7 on long-horizon agentic loops.
  • Chain multiple agents in parallel for complex tasks — manage a team of AI specialists.
