Chapter 2

Inside a Neuron — The Math That Isn’t Scary

Memory Map
💡

The Big Idea

A neuron is just multiply, add, apply one rule. Every neural network — from a 4-parameter toy to GPT-4’s trillions — is built from this same atomic operation. The activation function is the secret ingredient that makes stacking layers worthwhile.

⚙️ How a Neuron Computes

  1. Receive Inputs

    x₁, x₂, x₃ — data from the world or from previous neurons

  2. Multiply by Weights

    w₁×x₁, w₂×x₂, w₃×x₃ — each weight controls how much that input matters

  3. Sum + Bias

    z = Σ(w×x) + b — the weighted sum is the neuron’s raw score

  4. Activation Function

    f(z) = output — a non-linear rule that decides what to pass forward

⚡ The Three Activation Functions

Property 🟢 ReLU 🟠 Sigmoid 🔵 Tanh
Rule max(0, z) 1/(1+e-z) tanh(z)

🎒 Key Concepts

⚖️
Weights
Control how much each input matters. Positive = amplify, negative = suppress. Learned during training.
⊕
Bias
The neuron’s “default mood.” Shifts the activation threshold left (positive) or right (negative).
🔄
Non-Linearity
Without activation functions, 100 layers collapse into 1. The “bend” from activation is what makes depth useful.
🚫
Linear Collapse
A linear function of a linear function is still linear. Stacking without activation = wasted layers.
📈
Vanishing Gradients
Sigmoid/tanh saturate at extremes, shrinking gradients near zero. ReLU avoids this for positive inputs.
🎯
One Neuron = One Classifier
A single neuron with sigmoid is a tiny binary classifier — same as Ch 1’s linear boundary, plus a probability curve.

📖 Key Terms

Weighted Sum (z)
All inputs × their weights, summed together, plus the bias. The neuron’s raw score.
ReLU
max(0, z) — keep positives, kill negatives. The default activation for hidden layers.
Sigmoid
Squashes any number into (0, 1). Used for binary yes/no outputs.
Tanh
Squashes into (-1, 1). Zero-centered, used in some recurrent networks.
Activation Function
A non-linear rule applied after the weighted sum. Creates the “bends” that make deep networks powerful.
Sparsity
When many neurons output zero (via ReLU), only relevant neurons “fire” — efficient and feature-selective.
Dead Neuron
A ReLU neuron stuck permanently at zero — can happen if weights push z negative for all inputs.

❔ Quick Quiz

What are the four steps inside a neuron?

1) Receive inputs, 2) Multiply each by its weight, 3) Sum all products + bias to get z, 4) Apply activation function to get the output.

Why can’t you just stack linear layers without activation?

A linear function of a linear function is still linear — the entire stack collapses into a single layer. Activation introduces non-linearity, letting each layer create genuinely new patterns.

What does a negative weight mean?

It suppresses that input — a larger value on that input pushes the neuron’s output down, not up.

ReLU(‑3.7) = ?

0. ReLU kills all negative inputs to zero.

Why is sigmoid used for output layers but not hidden layers?

Sigmoid outputs a probability (0–1), perfect for binary classification. But it saturates at extremes, causing vanishing gradients that slow learning in deep hidden layers. ReLU avoids this.

📌 Key Takeaways

  • A neuron = multiply × weights, add bias, apply activation. That’s the entire operation.
  • Weights control importance (positive = amplify, negative = suppress). Bias shifts the threshold.
  • ReLU is the default: fast, simple, avoids vanishing gradients. max(0, z).
  • Sigmoid squashes to (0,1) — great for binary output, bad in hidden layers.
  • Without activation, layers collapse. Non-linearity is what makes depth useful.
  • One neuron = one decision boundary. Complex problems need many neurons → layers (Chapter 3).
  • Same math at any scale: 4 parameters or 1.8 trillion — multiply, add, activate.
