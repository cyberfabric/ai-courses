The Big Idea

Attention is only one of four operations in a transformer block. Residual connections create gradient highways, layer norm keeps numbers stable, and the FFN processes what attention gathered. Stack 96+ blocks and you get GPT-level capability.

🎒 Core Concepts

🔀
Residual Connection
output = input + transform(input). Preserves the original, lets each operation learn refinements. Creates gradient highways.
📏
Layer Normalization
Normalize to mean=0, variance=1, then apply learnable scale (γ) and shift (β). Keeps numbers stable across 96+ layers.
⚙️
Feed-Forward Network
Two linear layers with ReLU: expand 4×, process, compress back. Holds 2/3 of block parameters. Stores factual knowledge.
📍
Positional Encoding
Adds a unique position “fingerprint” to each token’s embedding before the first block. Without it, word order is invisible.
🚫
Causal Mask
Sets future positions to -∞ before softmax → 0 attention weight. Tokens can only see the past. Essential for generation.
📊
Parameter Budget
~12d² params per block. FFN = 8d² (2/3), Attention = 4d² (1/3). GPT-3: 96 blocks × 1.81B ≈ 175B total.

📏 Layer Normalization in 4 Steps

  1. Compute Mean

    μ = sum of values / count. For [0.70, 0.20, 1.10, 0.50]: μ = 0.625.

  2. Center (subtract mean)

    [0.075, -0.425, 0.475, -0.125]. Values now centered around zero.

  3. Compute Variance & Divide

    σ ≈ 0.327. Divide centered values by σ. Result: [0.23, -1.30, 1.45, -0.38].

  4. Scale & Shift

    Multiply by learnable γ, add learnable β. Model discovers optimal scale for each dimension.

📖 Key Terms

Residual Connection
Add the input back to the output: out = in + transform(in). Prevents vanishing gradients.
Layer Normalization
Normalize to mean=0, variance=1, then scale/shift with learnable parameters.
Feed-Forward Network (FFN)
Two linear layers with ReLU: expand 4×, activate, compress back. Processes each token independently.
Positional Encoding
Vectors added to embeddings encoding word position. Sinusoidal (original) or RoPE (modern).
Causal Mask
Triangular mask setting future positions to -∞. Ensures tokens only attend to past context.
Decoder-Only
Architecture using causal attention only. Used by GPT-4, Claude, LLaMA. The dominant architecture today.
Transformer Block
One unit: Attention → Add&Norm → FFN → Add&Norm. Models stack 12–120+ of these.

❔ Quick Quiz

Why do we need residual connections?

They create “gradient highways” that let error signals flow directly to early layers. Without them, gradients vanish across 96+ layers and early layers stop learning. They also let each operation learn refinements rather than full reconstructions.

Which component has more parameters: attention or the FFN?

The FFN. With 4× expansion, FFN has 8d² parameters vs. attention’s 4d². The FFN is 2/3 of each block’s parameters.

What happens without the causal mask during training?

The model would “cheat” by seeing future tokens. It could look at the answer instead of predicting it, learning nothing useful for generation.

Why is positional encoding necessary?

Attention treats all tokens identically regardless of position. Without positional encoding, “the cat sat” and “sat cat the” would be indistinguishable.

What does layer normalization guarantee about its output?

Mean ≈ 0 and variance ≈ 1. This keeps values in a stable range, preventing numerical explosion or collapse across many layers.

📌 Key Takeaways

  • One block = 4 operations: Attention → Add&Norm → FFN → Add&Norm.
  • Residual connections preserve the input and create gradient highways for deep learning.
  • Layer norm keeps numbers stable (mean=0, var=1) across 96+ layers.
  • FFN expands, ReLU-filters, compresses. Holds 2/3 of parameters and stores factual knowledge.
  • Positional encoding is added before block 1 so the model knows word order.
  • Causal mask hides future tokens — essential for autoregressive generation.
  • Decoder-only (GPT, Claude, LLaMA) is the dominant architecture today.
