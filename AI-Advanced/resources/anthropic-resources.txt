## Engineering Blog Posts

4. Effective Context Engineering for AI Agents
   URL: https://www.anthropic.com/engineering/effective-context-engineering-for-ai-agents
   Topics: Context vs prompt engineering, context rot, token curation, system prompts,
           runtime retrieval, "just in time" strategies, compaction, sub-agent architectures
   Summary: Defines context engineering as "curating and maintaining the optimal set of
   tokens during LLM inference." Key insight: treat context as a scarce resource,
   find the smallest set of high-signal tokens.

## Research Papers

9. How AI Assistance Impacts the Formation of Coding Skills
   URL: https://www.anthropic.com/research/AI-assistance-coding-skills
   Topics: RCT with 52 engineers, skill atrophy, AI interaction modes
   Key Finding: Engineers using AI scored 17% lower on comprehension quizzes,
   BUT those who used AI strategically (asking questions, seeking explanations)
   retained more knowledge than pure delegators.
   Relevance: Chapter 3 (Mindset Shift), Chapter 11 (Code Review)

10. How AI Is Transforming Work at Anthropic
    URL: https://www.anthropic.com/research/how-ai-is-transforming-work-at-anthropic
    Topics: Productivity metrics (50% boost), skill expansion, full-stack development,
            mentorship reduction concerns
    Key Finding: 60% of work involves Claude. Engineers becoming more "full-stack."
    Concerns about long-term skill development.
    Relevance: Chapter 1 (New Paradigm), Chapter 3 (Mindset Shift)

11. AI's Impact on Software Development (Anthropic Economic Index)
    URL: https://www.anthropic.com/research/impact-software-development
    Topics: Automation rates (79% agents vs 49% chatbots), UI/frontend dominance (59%),
            startup vs enterprise adoption (33% vs 13%)
    Key Finding: Claude Code enables substantially more automation than chatbot interfaces.
    Frontend work dominates AI-assisted development.
    Relevance: Chapter 2 (AI Coding Landscape), Chapter 9 (Agentic Workflows)
